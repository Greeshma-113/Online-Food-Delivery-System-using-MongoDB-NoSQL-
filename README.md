The Online Food Delivery System is a MongoDB-based NoSQL database project designed to simulate the functionality of popular food delivery platforms such as Swiggy and Zomato. The system manages customers, restaurants, menu items, orders, payments, delivery agents, and customer reviews through separate MongoDB collections.

This project demonstrates the implementation of MongoDB CRUD operations, filtering, sorting, aggregation pipelines, grouping, and advanced analytical queries. Customers can browse restaurants, view menus, place food orders, make payments, track delivery status, and provide ratings and reviews. Restaurant and payment data are analyzed using aggregation operations to generate useful business insights such as total revenue, popular payment methods, customer activity, restaurant ratings, and delivery performance.
